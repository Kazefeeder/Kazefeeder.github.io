css选择器专项笔记

发表于 2019-12-05 分类于 编程 ， python

response.css(‘a’) 选择名字为a的标签节点
response.css(‘a’).extract() 提取节点
response.css(‘a’).extract_first() 提取第一个节点

属性选择：

response.css(‘a[href=”image1.html”]’).extract() 提取名字为a ，含有属性 href=”image1.html”的标签，属性中中括号，属性内容双引号。

response.css(‘a[href=”image1.html”] img’).extract() 在上一个例子的基础上寻找符合条件的a标签下的img标签
response.css(‘a[href=”image1.html”] img::text’).extract()
response.css(‘a[href=”image1.html”] img::attr(src)’).extract()
提取内部文本和其他属性，文本直接双冒号加text，其他属性双冒号attr（属性名）这个属性名不用引号。
